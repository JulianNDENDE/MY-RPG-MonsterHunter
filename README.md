# MY-RPG-MonsterHunter
Création d'un jeu de type RPG sur le thème de Monster Hunter.  
Le projet à été fait en C avec la librairie CSFML.

# ![Screenshot du jeu](./assets/hud/Screenshot_20221017_133630.png?raw=true)

## Lancement du projet

*[CSFML REQUIS]*
```bash
make
# and
./my_rpg
```
